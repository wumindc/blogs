# 架构设计：基于 VitePress 的博客与教程系统

## 1. 目标
搭建一个高性能的、以 Markdown 驱动的纯静态网站，提供“基础教程”和“近期文章”（博客）两种不同心智体系的阅读体验。

## 2. 技术栈
- **核心框架**：VitePress (由 Vue 团队主导，基于 Vite 构建)
- **文档编写**：Markdown / MDX
- **包管理器**：Yarn
- **语言**：TypeScript (配置文件)

## 3. 目录结构设计
```text
.
├── docs/                # 文档源文件目录
│   ├── .vitepress/      # VitePress 配置目录
│   │   └── config.mts   # 核心配置文件
│   ├── blog/            # 博客分类文章
│   │   ├── index.md     # 博客首页
│   │   ├── ...          # 具体博文
│   ├── tutorial/        # 教程分类文章
│   │   ├── index.md     # 教程首页
│   │   ├── ...          # 具体教程
│   └── index.md         # 站点首页
├── package.json         # 项目依赖与脚本配置
└── CHANGELOG.md         # 变更记录
```

## 4. 路由与侧边栏划分
- `/tutorial/` 路径下的页面，侧边栏将呈现体系化的树形结构，方便读者一步步学习。
- `/blog/` 路径下的页面，侧边栏作为近期文章列表或时间轴使用。

## 5. 部署方案
开发人员可以在本地运行 `yarn docs:dev` 进行预览。上线时通过 `yarn docs:build` 生成产物后静态托管。
