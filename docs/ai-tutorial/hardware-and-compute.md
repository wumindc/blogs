# 什么是算力与硬件？

> “算力即权力 (Compute is Power)”。在生成式 AI 统治硅谷的时代，算力不仅决定了模型能不能跑起来，更是大国博弈的核心资产。

如果你是传统的业务开发，你可能习惯了云服务器上几核几 G 的 CPU 和内存指标。但到了 AI 时代，主角变成了“算力卡（Accelerator）”。

## 1. 为什么 CPU 不能搞大模型？

CPU（中央处理器）被称为计算界的“老教授”。它非常聪明，能处理各种极其复杂的控制逻辑（如 `if-else` 分支、线程上下文切换、中断处理）。
但问题是，CPU 里的运算核心（Core）很少，通常只有几十个。

当你训练或者运行一个大模型时，你面对的是数百亿个类似于 `y = wx + b` 这样简单的矩阵乘加运算（矩阵点积运算）。如果让一个老教授一笔一划去算一百亿道口算题，他会累死。

此时你需要的是“成千上万个只会做基本算术的小学生”同时算——这就是 **GPU（图形处理器）**。

## 2. GPU 与其它 AI 芯片 (TPU / NPU)

### GPU (Graphics Processing Unit)
GPU 其实最初是为了打游戏时渲染屏幕像素发明的。因为它拥有成千上万个流处理器核心，天然就是**大规模并行计算**的王者。
* **NVIDIA (英伟达)**：当今 AI 硬件绝对霸主。其推出的 **A100** 和更新的 **H100 / Blackwell** 芯片是目前各大互联网巨头买断货的基础“硬通货”。
* **CUDA 生态**：英伟达之所以无敌，不仅仅是芯片造得好，更是因为它提供了一套名为 CUDA 的并行计算软件接口。几百万 AI 研究员只能在这个生态上写代码，这成了它最强悍的护城河。

### TPU (Tensor Processing Unit)
* 这是 Google 专门为深度学习（特别是它的 TensorFlow 框架）定制的专用芯片。由于砍掉了图形渲染等没用的功能，专注于“张量 (Tensor)”运算，在自家数据中心里性能极佳。

### NPU (Neural Processing Unit)
* 如今，从苹果手机的 A 系列芯片、M 系列芯片，到各类国产高端芯片，都内嵌了 NPU（神经网络引擎）。这种芯片直接将神经网络最底层的乘加指令固化在物理电路上，用于在你手机本地超低功耗地跑人脸识别或者边缘端大模型。

## 3. 算力的单位：FLOPS

我们经常能看到新闻里吹牛某某机房的算力达到了多少“P”。这里的单位是 **FLOPS (Floating Point Operations Per Second，每秒浮点运算次数)**。

* **TeraFLOPS (TFLOPS)**：每秒一万亿次。一张顶配家用显卡 RTX 4090 大约在这个量级。
* **PetaFLOPS (PFLOPS)**：每秒一千万亿次。通常中型 AI 集群。
* **ExaFLOPS (EFLOPS)**：百亿亿次级别的国家级超算中心。

> **小节总结**：大模型参数突破千亿级别（百 GB 级别）。单张顶配显卡（例如 80GB 显存的 A100）根本连模型的脚趾头都塞不下。这就引出了下一节：无数张独立显卡是如何拼起来干活的？
